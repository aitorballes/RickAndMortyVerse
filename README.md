# Rick And Morty Characters App

![SwiftUI](https://img.shields.io/badge/SwiftUI-%23FF5722.svg?style=for-the-badge&logo=swift&logoColor=white)

## Description

This application, developed in **SwiftUI**, consumes the [Rick And Morty API](https://rickandmortyapi.com/) to display a list of all characters from the series. It allows filtering by **name**, **gender**, **species**, and **status**. When selecting a character, a detail view is shown with relevant information, and users can **save it as a favorite**, using **Core Data** for data persistence.

## Features

- **Character list** with filtering options.
- **Character detail view** with relevant information.
- **Favorites**: option to save and persist characters on the device.
- **MVVM Architecture**.
- **Unit Tests and UI Tests** implemented.

## Technologies Used

- **SwiftUI**
- **Combine**
- **Core Data**
- **MVVM** (Model-View-ViewModel)
- **XCTest** (Unit Tests and UI Tests)
